# MeTube

A YouTube clone made with React and [YouTubeV3](https://rapidapi.com/ytdlfree/api/youtube-v31) API.

## Requirements

- npm >= 10.7.0

- node >= 20.12.2

---

## Installation

Open `cmd` or `bash` in the project directory.

```bash
npm install
```

## Run Locally

```bash
npm run dev
```
